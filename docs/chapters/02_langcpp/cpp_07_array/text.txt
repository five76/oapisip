Константное выражение определяет размер массива, т. е. числе
элементов этого массива. Например, согласно описанию
i n t A[10];
объявлен массив с именем А, содержащий 10 элементов целого
типа. Элементы массива обозначаются индексированными имена­
ми. Нижнее значение индекса равно 0:
А[0],
А[8],
А[1],
А[9]
А[2],
А[3],
А[4],
А[5],
А[6],
А[7],
В отличие от Паскаля в Си нельзя определять произвольные
диапазоны для индексов. Размер массива, указанный в описа­
нии, всегда на единицу больше максимального значения ин­
декса.
Размер массива может явно не указываться, если при его
объявлении производится инициализация значений элементов.
Например:
int
p[]={2,
4,
б,
10,
1};
В этом случае создается массив из пяти элементов со следую­
щими значениями:
р[0]=2,
р[1]=4,
р[2]=б,
р[3]=10,
р[4]=1
В результате следующего объявления массива
int
М[б]={5,
3,
2};
будет создан массив из шести элементов. Первые три элемента
получат инициализированные значения. Значения остальных бу­
дут либо неопределенными, либо равны нулю, если массив вне­
шний или статический.
Рассмотрим несколько примеров программ обработки одно­
мерных массивов.
